# **App Name**: Free Agent Connect

## Core Features:

- Landing Page Sections: Sections for About, Who We Are, Services, and Contact information.
- Smooth Scrolling: Smooth scrolling functionality when clicking on menu items.
- Freelancer Profiles: Display freelancer profiles with skills, experience, and rates.
- Contact Form: A contact form for potential clients to reach out.
- AI Freelancer Match: Generative AI tool to match clients with suitable freelancers based on project requirements. The tool can determine which aspects of each project requirements should weigh more to find more suitable freelancers.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for trust and professionalism.
- Background color: Light grey (#F5F5F5) for a clean, modern feel.
- Accent color: Teal (#009688) for highlighting key information.
- Body and headline font: 'Inter', a grotesque-style sans-serif known for its clean, modern look suitable for both headlines and body text.
- Use Material Design icons to maintain consistency with the overall design.
- Single-page layout with clear sections for easy navigation.
- Subtle animations for transitions and interactions, adhering to Material Design principles.
